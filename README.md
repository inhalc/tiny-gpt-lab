# Tiny GPT Lab

A lightweight **research bootcamp** workspace for GPT/LLM learning-by-doing.

## Goal
This repository contains workflow, code, experiments, and notes for a small GPT research bootcamp.

Primary goals:
- Understand how GPT-style models work from first principles
- Implement key components (eventually transformers) from scratch
- Build strong research workflow habits
- Develop debugging and experimentation skills

## Weekly Workflow
Every week follows the same rhythm:
- **Monday**: task kickoff and scope
- **Midweek**: self-exploration and experiments
- **Friday**: discussion, debugging, and review

## Lab Rules (Culture)
- Every experiment must be logged
- Every submission requires a PR
- Every result should include observations
- Understanding is more important than just running code

## Repository Structure
- `docs/`: process documentation and templates
- `week01_bootstrap/`: first-week onboarding tasks and starter code
- `shared_notes/`: ongoing notes and references
- `presentations/`: slide decks or discussion material

## Current Focus
Week 1 is **Lab Bootstrap Week**: environment, workflow, basic training loop, and scientific reporting habits.
