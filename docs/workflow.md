# git / PR

Branch：

```bash
git checkout -b week1-bootstrap
```

Commit：

```bash
git add week01_bootstrap/workspace
git commit -m "week1 bootstrap"
```

Push：

```bash
git push -u origin week1-bootstrap
```

PR：

- base：`main`
- branch：`week1-bootstrap`
- title：`Week 1 Bootstrap`

写：

- 跑了什么
- 改了什么
- 看到什么
- 卡在哪里
